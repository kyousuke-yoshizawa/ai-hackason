# 発表スライド（Design deck版）

Claude の design（スライド作成）ツールで作成した、どうぶつの森風の発表スライドです。
`docs/presentation/presentation.md`（Marp形式）とは別の、もう1つの発表資料候補です。

## ファイル構成

- `おでかけの森_発表スライド.dc.html` — 15枚構成。アプリ名は現行どおり「**ことこと町**」表記。
- `おでかけの森_発表スライド_19枚版.dc.html` — 19枚構成（Vision/Goal/工夫の深掘りスライドを追加）。アプリ名は「**おでかけの森**」表記。
- `deck-stage.js` / `image-slot.js` / `support.js` — スライドの表示・スクリーンショット差し込み用の補助スクリプト（自動生成、直接編集しない）。
- `-2026-07-23-155106-mrx5mq9g-j0ig.png` / `pasted-1784789514641-0-mrx5lfuw-xbp7.png` — 差し込み用の画像素材（内容は同一）。

## ⚠️ 注意：2つのファイルでアプリ名表記が異なります

15枚版は「ことこと町」、19枚版は「おでかけの森」とアプリ名が異なります。どちらか一方に統一するか、正式名称を確認してから使ってください（現在のコードベース・CLAUDE.mdでは「ことこと町」表記です）。

## 見方

`.dc.html` ファイルをブラウザで直接開くと、`support.js` が読み込まれてスライドショーとして表示されます（矢印キー / PageUp・PageDown / タップで送り）。画像プレースホルダー（`<image-slot>`）は、対応するビューア上でドラッグ&ドロップして差し替えられます。

編集する場合は、claude.ai の design プロジェクト側（プロジェクト名「おでかけの森 発表スライド」）で編集するのが確実です。このリポジトリ内のファイルは静的なコピーのため、直接HTMLを編集した変更はdesignプロジェクト側には反映されません。
