# ことこと町 世界観ネーミング 商標・実在名チェック

関連Issue: [#152](https://github.com/kyousuke-yoshizawa/ai-hackason/issues/152)

## 調査目的

要件定義書 `outing-plan-ai-requirements-v2.md` 7章「世界観の一貫性（新規）」の非機能要件
「架空の店舗名・地名が実在の企業名・商標と偶然一致しないか確認する」について、
未実施だった確認作業を行い、結果を記録として残す。7/25デモ発表で権利関係を指摘された
際の備えとする。

## 調査手法（注意）

- **Web検索代替**: 本来はJ-PlatPat（特許情報プラットフォーム、
  https://www.j-platpat.inpit.go.jp/）での商標の公式検索が望ましいが、本タスクでは
  直接のツールアクセスができないため、WebSearchツールによる「[名称] 商標」等のクエリ
  検索で、登録商標の存在が広く知られているか（商標登録情報が検索結果に表れるか）を
  確認する形で代替した。**J-PlatPatでの公式な商標番号レベルの照合ではない**ため、
  正式な商標クリアランスが必要な場合は別途弁理士等による公式検索を推奨する。
- **実在店舗確認**: 各名称について「[名称] 店舗」「[名称] カフェ」等のクエリで日本語
  Web検索を行い、同名の実在店舗・チェーン・施設の有無を確認した（2026-07-16実施）。
- **判定基準**:
  - **OK**: 同名の実在店舗が見つからない、または個人店が1軒程度見つかる程度（偶然の
    一致として許容）
  - **要注意**: 直接の完全一致ではないが、著名なブランド・プロジェクトと語感・構成が
    近く、混同の可能性が否定できない
  - **NG**: 全国チェーン・著名店・近隣地域の実在店と紛らわしい、または実在の企業が
    同一名称で現に営業している
- 任天堂等の作品固有名詞（ゲーム・アニメのキャラクター名・地名）との混同がないかも
  あわせて確認した。

## 改名決定（2026-07-16、吉沢承認）

### ことりカフェ（NG）→ ことりの休憩処 に改名

実在の全国チェーン「株式会社ことりカフェ」（鳥カフェ、上野本店・心斎橋店等）と完全一致するため改名。
小鳥・のんびりした雰囲気を維持しつつ「カフェ」という単語自体を避けた。

- 案1（採用）: ことりの休憩処（ことりのきゅうけいどころ）
- 案2（不採用）: さえずり茶房（さえずりさぼう）

### ちいさな本の森（要注意）→ ひなた文庫 に改名

著名プロジェクト「こども本の森」（安藤忠雄氏、大阪・岩手・愛媛で展開）と語感が近いため、念のため改名。
「本の森」という語を避け、世界観の「ことこと町」らしい温かみを保持した。

- 案1（採用）: ひなた文庫（ひなたぶんこ）
- 案2（不採用）: 小さな読書小屋（ちいさなどくしょごや）
