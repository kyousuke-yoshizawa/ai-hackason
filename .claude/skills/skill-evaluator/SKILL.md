# Skill Evaluator

このスキルは Claude Code 内で使用されているスキルを評価・テストするためのツールです。既存スキルの性能、機能、品質を測定し、詳細な評価レポートを生成します。

## 評価機能

### 1. スキル性能の測定
- **実行時間の計測**：スキル実行にかかった時間を測定
- **応答性能分析**：パフォーマンスメトリクスを分析
- **エラー発生率**：エラーの発生頻度を計測

### 2. スキル機能の検証
- **期待結果の確認**：スキルが期待した結果を返すか確認
- **テストケース実行**：複数のテストシナリオで機能を検証
- **機能完全性チェック**：ドキュメントに記載された機能がすべて動作するか確認

### 3. スキル品質の分析
- **ユーザーインタフェース評価**：使いやすさとインタラクティブ性を評価
- **ドキュメント明確性**：説明文やヘルプの明確性を検証
- **トリガー条件検証**：スキルが適切な条件でトリガーされるか確認

### 4. 評価結果のレポート生成
- **詳細レポート**：評価結果を構造化されたレポートで表示
- **パフォーマンス分析**：メトリクスを視覚化
- **改善提案**：自動生成された改善案を提供

## 使用方法

```
/skill-evaluator
```

実行すると、以下のステップで評価を進めます：

### ステップ 1: 評価対象スキルの選択
`ListSkills` MCP ツールを使用して、利用可能なスキル一覧を表示します。
ユーザーは評価対象スキルを選択します。

### ステップ 2: 評価項目の選択
以下から評価項目を選択します：
- **性能評価**：実行時間、応答性能、エラー率
- **機能評価**：期待結果との一致、テストケース成功率
- **品質評価**：UI/UX、ドキュメント、トリガー条件

### ステップ 3: テストケースの入力
評価対象スキルに提供するテストケースまたは評価基準を入力します：
- スキルへの入力パラメータ
- 期待される出力
- 評価基準

### ステップ 4: スキル実行とデータ収集
- 評価対象スキルを実際に実行
- パフォーマンスメトリクスを収集
- 実行結果を記録

### ステップ 5: 結果分析とレポート生成
- 収集したデータを分析
- 評価スコアを計算
- 改善提案を生成
- 構造化されたレポートを作成

## 実装の詳細

### 使用する MCP ツール
- `ListSkills` - 評価対象スキル一覧の取得
- 対象スキルの実行（Skill ツール）

### 対話的フロー
`AskUserQuestion` を使用して、以下の段階的な質問を実施：
1. 評価対象スキルの選択
2. 評価項目の選択
3. テストケースの入力
4. 詳細度の選択

### レポート形式
評価レポートには以下の情報を含めます：

```
【スキル評価レポート】
スキル名: [対象スキル]
評価日時: [実行日時]

【性能評価】
- 平均実行時間: [XXX ms]
- 最大実行時間: [XXX ms]
- エラー発生率: [XX%]
- 成功率: [XX%]

【機能評価】
- テスト成功数: [XX/YY]
- 期待結果との一致度: [XX%]
- 実装漏れ: [機能リスト]

【品質評価】
- UI/UX スコア: [X/10]
- ドキュメント明確性: [X/10]
- トリガー適切性: [X/10]

【総合評価】
- 総合スコア: [XX/100]
- 評価レベル: [優秀/良好/標準/改善必要]

【改善提案】
1. [提案1]
2. [提案2]
3. [提案3]

【詳細分析】
[詳細な分析結果]
```

## 評価基準

### 性能評価基準
- 優秀：50ms 以下
- 良好：50-200ms
- 標準：200-500ms
- 改善必要：500ms 以上

### 機能評価基準
- 優秀：95% 以上成功
- 良好：80-95%
- 標準：60-80%
- 改善必要：60% 未満

### 品質評価基準
- 各項目を 1-10 で評価
- 8 点以上：優秀
- 6-8 点：良好
- 4-6 点：標準
- 4 点未満：改善必要
